# CoVibe (v1)

Status: CoStaged_v1
LastUpdatedUTS: SESSION_UTS_PLACEHOLDER

---

## 1. Short definition

CoVibe is the declared tone of a session or asset.

It is described using:

- CoVibe - plain language description of the tone.
- CoVibeLevel - intensity scale from 1 to 3.
- CoVoice - who the session feels like it is speaking as.
- CoStyle - language and structure style.
- CoBrand - which CoSuite brand context applies.
- CoTerm - which Co-terms are expected in play.

---

## 2. Example block

```text
[CoVibe]
CoVibe: calm and constructive
CoVibeLevel: 2
CoVoice: steward-mentor
CoStyle: minimal, direct
CoBrand: CoSteward ops
CoTerm: CoMetaEvolution, CoContrast, CoBPOE, CoVibe
```

---

## 3. Role in GIBindex

- Helps future AIs and humans understand how to read an asset.
- Supports filtering and routing by tone.
- Can be cross-linked with CoMetaEvolution and CoContrast when tone matters for safety.
