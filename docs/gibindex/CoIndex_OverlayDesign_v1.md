## 1. Purpose

Define how GIBindex provides a human-facing semantic overlay
for CoIndex’s AI-facing CoPortal surfaces.

GIBindex is a dual-language system:
- Human-readable labels, lenses and explanations
- AI-readable Gibber/Gibberlink structures

## 2. Overlays

### 2.1 Concept Lens
Shows:
- definition
- domain lineage
- human examples
- CoNuggets attached
- cross-repo references (CoSteward, CoCore, CoEthics)

### 2.2 Intent Lens
Shows:
- why the intent exists
- transformation or pressure direction
- civic/ethical constraints
- examples in context

### 2.3 Drift Overlay
- semantic drift heatmap
- pending proposals
- night-job findings
- contributor discussions

### 2.4 Reputation Overlay
- MeritRank distribution
- CoHalo contributions
- RepTag bundle
- ScriptTag status if applicable

## 3. Integration Rules
- GIBindex never overrides CoIndex
- It interprets and narrativizes only
- All cross-links remain bi-directional
